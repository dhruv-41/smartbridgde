Housing Trends Website

1. Add images in the images folder:
   - hero.jpg (for homepage background)
   - building.jpg (for about page)

2. Add your Tableau links:
   - dashboard.html → replace PASTE_TABLEAU_DASHBOARD_LINK_HERE
   - story.html → replace PASTE_TABLEAU_STORY_LINK_HERE

3. Open index.html in your browser to test locally.

4. Upload the whole folder to GitHub and enable GitHub Pages to deploy.
